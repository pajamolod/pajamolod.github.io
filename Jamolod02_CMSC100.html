Projects

Rental Management System

This terminal-based rental management system, coded in C, allows users to efficiently manage apartment units and tenants. It provides functionalities to add, remove, and edit tenants, search for occupants, move tenants between units, and print master lists. The system ensures proper organization by sorting units and tenants alphabetically while preventing duplicate entries.

Rshiny Diet Problem Solver

This Shiny application allows users to select food items and optimize their diet based on nutritional constraints and cost minimization. Using the Simplex method, the app constructs an objective function, applies constraints, and determines the most cost-effective meal plan while ensuring nutritional requirements are met. The solution includes a breakdown of selected foods, servings, and cost analysis.

Pest vs Plants: Farm Simulator

A text-based farming simulation game where players grow crops, manage farmland, and make decisions to handle pests and weather conditions. The game features randomized events, a farmland layout system, a scoring system, and a global leaderboard to track performance, coded in Python.
